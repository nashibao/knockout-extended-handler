ABOUT

Common JS Plugins(like jquery-ui, bootstrap-js) are written in script code block.

Knockout-extended-handler makes these plugins to be written in html block.

--before.--

$('#id').hoge({option: value})

--after.--

data-bind="ext:{hoge :{option: value}}"

A CASE

Knockout-extended-handler can be used for jquery-ui style plugins(introduced above) without additional settings.

for exam...

bootstrap-js

data-bind="ext:{tooltip:{title: 'title'}}"
data-bind="ext:{popover:{title: 'title', content: 'content'}}"
tooltip
popover

jquery-ui

data-bind="ext:{datepicker:{}}"

pickadate

data-bind="ext:{pickadate:{}}"

bootstrap-select

data-bind="ext:{selectpicker:{}}"
